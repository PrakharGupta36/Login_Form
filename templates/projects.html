Automated Monitoring System

A IoT, DS-ML based solution for Energizing the sustainable world affordably.

It will be integrated with Pollution Monitoring Sensors(PMS) and Weather Monitoring Sensors(WMS).

Pollution Monitoring Sensors(PMS)

  • Humidity
  • Parts Per Million (PPM2.5 & PPM10)
  • CO2, CO
  • SO2
  • NxOy

Weather Monitoring Sensors(WMS)

  • Temperature
  • Humidity
  • Solar Illumination

It will send these info at run time through the GPRS along with the GPS location.

ACP Monitoring

An IoT based solution for smooth functioning of railways.

It will store:

  • The count of times the Alarm chain Pulling (ACP) has been triggered.
  • The GPS location where the ACP was triggered.
  • The time when the ACP was triggered.
  • Loss of running time of the train because of specific ACP event.

It will send these info at run time through the GSM attached.

AWMS - Automated Water Monitoring System

A IoT based solution for better water management in running trains.

It will inform upcoming stations regarding water availability in the water tank storage in each coach.

It will send these info at run time through the GSM attached.
